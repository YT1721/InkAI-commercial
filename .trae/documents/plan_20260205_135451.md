# 下一步开发计划：画布交互深度优化

我们已经完成了数据管理和核心节点功能的增强。为了让 InkAI 的画布操作达到像 Figma 或 Miro 那样的专业水准，接下来的重点将放在**画布交互的细腻度**上。

## 🛠 拟实施的功能

1.  **实时框选 (Real-time Selection)**
    *   **现状**：目前只有在松开鼠标后才知道选中了哪些节点。
    *   **优化**：在拖拽选择框的过程中，实时计算并高亮（虚线框或发光效果）被框选的节点，给用户即时的视觉反馈。

2.  **连线交互增强 (Connection Management)**
    *   **现状**：删除连线不够直观（目前主要靠删除节点）。
    *   **优化**：
        *   当鼠标悬停在连线上时，连线变粗并高亮。
        *   支持点击选中连线，并按下 `Delete` 或 `Backspace` 键直接删除。
        *   优化贝塞尔曲线的路径算法，避免连线穿过节点中心。

3.  **多节点批量操作**
    *   **优化**：支持框选多个节点后进行批量移动、批量删除。
